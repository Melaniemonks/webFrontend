Forms

  • Forms are ways we interact with a web page. It uses the form element. This element uses controls such as input fields, selects menus and buttons.
    1. Action: this attribute is the URL that the form will be submitted to so it can be processed on server side.
    2. name: This is an attribute that can be found in the "input" or "form" element. This is used to access the information inside it
  • Accessing a form: document.forms[] returns an HTML collection of all the forms in the document in the order they appear.

Object Oriented Programming

This is a style of programming that involves separating the code into objects that have properties and methods.

  • An Introduction To OOP: This is used to represent objects in the real world
    1. Encapsulation: In this case, The inner workings are kept hidden inside the object. Just the front end is exposed to the user.
    2. Polymorphism: The same process can be used for different objects
    3. Inheritance: This is taking the features of one object and adding new features. We take objects that already exists and inherit its properties and methods
    4. Classes: This is used to define the blueprint for an object
  • Constructor Functions: This function uses the "this" keyword to represent the object that will be returned by the constructor function.
