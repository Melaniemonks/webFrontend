Further Functions

  1. The call() method can be used to set the value of 'this' inside a function
  2. A recursive function will continue to invoke itself until a certain condition is met

AJAX

  • AJAX is a technique that allows web pages to communicate asynchronously with a server. It dynamically updates web pages
  • The internet is divided into 'Clients' and 'Servers'
  • Fetch API is used to request and send data asynchronously across a network.
